דוח הפרויקט, שם מסוכם הכל בהרחבה:
pdf.פרויקט רשתות תקשורת מחשבים

כל הקבצים משתייכים לשני החלקים של הפרויקט

חלק ראשון:
* liran_dns_input.csv - הקובץ שהמחברת טוענת
* dns_input_capture.pcap - CSV קובץ הלכידה בווירשארק שדרכו הכנתי בעצמי את הקובץ 
* tcpip_project_jupyter_notebook.ipynb - זו מחברת הג'ופיטר לאחר הרצה
* jupyter_send_demo_capture.pcap - קובץ הלכידה של הודעות הדמה ששולחת המחברת
* jupyter_send_csv_capture.pcap - ששולחת המחברת CSV קובץ הלכידה של הודעות ה

חלק שני (הסבר איך להריץ רשום בדוח, יש בו תוכן עניינים):
* server_interface - הקוד של אפליקציית השרת
* client_interface - הקוד של אפליקציית הלקוח
* 2_clients_chat.pcap - קובץ לכידה של שיחה לדוגמה בין שני משתמשים
* 5_clients_chat.pcap - קובץ לכידה של שיחה לדוגמא בין חמישה משתמשים
